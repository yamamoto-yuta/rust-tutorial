# rust-tutorial

Usage:
```
$ git clone <REPOSITORY_URL>
$ docker-compose up -d
$ docker exec -it <CONTAINER_ID> bash
```

helloworld:
```
$ cargo new helloworld
$ cd helloworld/
$ cargo run
```
