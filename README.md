# rust-tutorial
- RUSTを勉強するためのリポジトリ

教材：[The Rust Programming Language 日本語版](https://doc.rust-jp.rs/book-ja/title-page.html)


環境構築：
```
$ git clone <REPOSITORY_URL>
$ docker-compose up -d
$ docker exec -it <CONTAINER_ID> bash
```

helloworld：
```
$ cargo new helloworld
$ cd helloworld/
$ cargo run
```
